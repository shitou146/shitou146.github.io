实验报告

简介

运用Git Bash指令中的mv指令,tar -xf指令和rm指令对文件进行移动,解包以及删除操作

命令

  • 第一步：下载文件到个人文件夹
  • 第二步:用tar -xf命令来解压videoClips包文件
  • 第三步：删除下载的包文件，并列出解包的文件

指令如下

  • 231 cd /
  • 232 cd /d/桌面
  • 233 ls
  • 234 mv videoClips /d/Git/shitou
  • 235 cd /
  • 236 cd /d/shitou
  • 237 cd /d/Git/shitou
  • 238 ls
  • 239 tar -xf videoClips
  • 240 ls
  • 241 rm videoClips

总结

经过本次的实验操作,初步掌握相应的Git Bash命令来进行一系列文件操作,包括解包,删除包文件和列举解包后的文件。
